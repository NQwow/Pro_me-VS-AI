# 文字冒险游戏 - AI 剧情生成版

一个基于 Python 的命令行文字冒险游戏。玩家可以创建角色，经历多回合的剧情选择，最终根据属性值获得不同结局。未来计划接入 DeepSeek 等 AI 接口，实现动态生成的剧情和选项。

---

## 功能特点

- **角色创建**：支持随机生成或手动输入四项属性（0-100）
- **多回合冒险**：默认 5 回合，每回合提供两个选项，影响角色属性
- **结局系统**：根据最终属性值判定结局类型
- **记忆系统**（规划中）：保存历史事件，让剧情更具连续性
- **数据持久化**（规划中）：保存游戏结局，支持历史回顾
- **AI 集成**（规划中）：调用大语言模型动态生成剧情与选项

---

## 安装与运行

### 环境要求
- Python 3.7+
- 依赖库：无（仅使用标准库）

### 运行步骤
1. 克隆仓库
   ```bash
   git clone https://github.com/NQwow/PYGame.git
   cd PYGame
   ```
2. 直接运行游戏
   ```bash
   python main.py
   ```
